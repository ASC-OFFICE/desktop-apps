ONLYOFFICE 4.4 - 12 июля 2017

Новые функции

Редактор документов

  • Добавлена функция Специальная вставка;
  • Добавлена поддержка форматированного текста;
  • Добавлены переводы стилей абзацев;
  • Добавлено отображение комментариев в режиме просмотра;
  • Добавлена возможность скрывать решенные комментарии;
  • Добавлена возможность задавать настраиваемые колонки.

Редактор электронных таблиц

  • Добавлена поддержка функций с условным форматированием;
  • Добавлена возможность задавать пользовательский разделитель для импорта/экспорта CSV-файлов;
  • Добавлена поддержка функций ACOT, ACOTH, ARABIC, BASE, BETA.DIST, BETA.INV, BETADIST, BINOM.DIST, BINOM.INV, CEILING.MATH, CEILING.PRECISE, CHIDIST, CHIINV, CHISQ.DIST, CHISQ.DIST.RT, CHISQ.INV, CHISQ.INV.RT, COMBINA, CONCAT, CONFIDENCE.NORM, CONFIDENCE.T, COT, COTH, CSC, CSCH, DECIMAL, EXPON.DIST, F.DIST, F.DIST.RT, F.INV, F.INV.RT, FDIST, FINV, FLOOR.MATH, FLOOR.PRECISE, GAMMA, GAMMA.DIST, GAMMA.INV, GAMMA.PRECISE, GAMMADIST, GAMMAINV, GAUSS, IFNA, IMCOSH, IMCOT, IMCSC, IMCSCH, IMSEC, IMSECH, IMSINH, IMTAN, ISO.CEILING, LOGNORM.DIST, LOGNORM.INV, NUMBERVALUE, PERCENTILE.EXC, PERCENTILE.INC, PERCENTRANK.EXC, PERCENTRANK.INC, RANK, RANK.AVG, RANK.EQ, SEC, SECH, T.DIST, T.DIST.2T, T.DIST.RT, T.INV, T.INV.2T, TDIST, TINV, XOR;
  • Скрытые строки теперь исключаются при копировании, автозаполнении, форматировании и т.д.;
  • Изменен цвет активной ячейки;
  • Изменено отображение закрепленной области: теперь у нее появилась тень;
  • Добавлены переводы стилей ячеек;
  • Добавлена возможность поиска и замены с использованием данных из выделенной области;
  • Добавлена возможность скрывать решенные комментарии;
  • Добавлена возможность задавать маркеры и нумерацию для текста внутри фигуры;
  • Добавлена возможность задавать колонки для областей с текстом;
  • Добавлена возможность добавлять соединительные линии для фигур.
  • Добавлена поддержка сочетаний клавиш Shift+Delete, Ctrl+Insert и Shift+Insert для операций вырезания/вставки.

Редактор презентаций

  • Добавлена возможность соединять фигуры с помощью линий;
  • Добавлена возможность задавать колонки для областей с текстом;
  • Добавлена возможность задавать маркеры и нумерацию для текста внутри фигуры;
  • Добавлена проверка орфографии;

Конвертация

  • Существенно повышена скорость открытия файлов OOXML;
  • Улучшена конвертация из/в формат RTF;
  • Улучшена поддержка защищенных паролем файлов OOXML и двоичных файлов Microsoft (DOC, XLS);
  • Добавлена поддержка файлов PPT, защищенных паролем.
